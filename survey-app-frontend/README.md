# Frontend App (React + Vite + Chakra UI)

## Setup Instructions

1. Use Node v20 (via `nvm`)

   ```bash
   nvm use 20
   ```

2. Install dependencies

   ```bash
   npm install
   ```

3. Start development server

   ```bash
   npm run dev
   ```

App will be available at [http://localhost:5173](http://localhost:5173)
